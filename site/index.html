Slash commands

Everything lives in Discord — no dashboard needed.

/setchannel
Pick the channel where price alerts are posted. Requires Manage Channels.
/track <item_id>
Start tracking a limited. Grabs the current price snapshot as a baseline.
/untrack <item_id>
Remove an item from the tracking list.
/list
Show all tracked items with their last known value and RAP.
/check
Manually pull current prices for every tracked item right now.

Get started in 3 steps

1

Add RoliWatch to your server

Click Add to Discord and authorise the bot. It needs Send Messages, Embed Links, and View Channel.

2

Set your alert channel

Run /setchannel #your-channel so RoliWatch knows where to post price alerts.

3

Track your limiteds

Find the item ID in the Rolimons URL — e.g. rolimons.com/item/1365767 — then run /track 1365767.
